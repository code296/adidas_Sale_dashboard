Adidas Sales Analysis Dashboard

A Power BI dashboard analyzing Adidas retail sales performance across regions, products, and retailers.


📊 Overview

This dashboard explores Adidas sales data to uncover trends in revenue, profitability, and regional/retailer performance. Built as part of my Data Analyst portfolio to demonstrate skills in data modeling, DAX, and dashboard design in Power BI.

🗂️ Dataset

The dataset (Data Sales Adidas) contains 9,648 records with the following fields:

Column	Description
Retailer	Retail partner name (e.g., Foot Locker, West Gear)
Retailer ID	Unique retailer identifier
Invoice Date	Date of sale
Region / State / City	Sales location hierarchy
Product	Product category
Price per Unit	Unit selling price
Units Sold	Quantity sold
Total Sales	Revenue generated
Operating Profit	Profit after costs
Operating Margin	Profit margin (%)
Sales Method	Channel (e.g., Outlet, In-store, Online)
